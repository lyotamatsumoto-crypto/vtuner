要約

最終更新: たった今 / 人格 / 理想スキーマ対応

VTuner 用の人格定義 JSON を作ってください。
次の条件を満たしてください。

- 少しクールで、でも身内にはやわらかい
- 視聴者には丁寧、配信者には少し近い
- 強い煽り、露骨な暴言は禁止
- 補助役寄りで、でしゃばりすぎない
- 口調は落ち着いた方向
- 好きな言い回しと避けたい言い回しを反映する
- 必須キーを省略しない
- JSON 以外の説明文を含めない
- 理想的な JSON 構造見本に沿って埋める

返す対象:
- 基本性格
- 視聴者向け態度
- 配信者向け態度
- 禁止表現
- 口調
- 語尾
- 好きな言い回し
