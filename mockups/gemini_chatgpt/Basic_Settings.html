キャラプロフィール

ここはキャラの素材を置く場所です。
ここで設定したことをそのまま毎回話すわけではありません。
どう答えるかの細かいルールは「Detailed Rules」で扱います。

キャラ方針

人格プリセットは、完成した人格そのものではなく、基本性格・対配信者態度・対視聴者態度・口調プリセットの初期束です。
お姉さん 落ち着き・面倒見・やわらかめの初期束
ツンデレ 距離感強め・反応に緩急のある初期束
クーデレ 静かめ・控えめ・内側にやわらかさを持つ初期束
妹 親しみ・甘え・距離近めの初期束
世話焼き 補助意識・気配り・助け舟寄りの初期束
小悪魔 軽いからかい・遊び感・反応強めの初期束
天然 やわらかさ・素直さ・少しずれた可愛さの初期束

生意気後輩 距離近め・軽口・生意気さを含む初期束

からかい上手 軽い茶化し・場回し・親しみ寄りの初期束
あざとい 見せ方強め・可愛げ重視の初期束
真面目委員長 規律感・説明力・丁寧さ重視の初期束
オタク寄り解説役 知識寄り・説明好き・補足多めの初期束
「ヤンデレ寄り」は初期MVPでは除外または慎重扱いとします。必要になった場合は Detailed Rules または AI / JSON Studio 側で段階的に追加します。
新しい人格プリセットを手動で正式追加する場合は「Detailed Rules」で行います。
外部AIで定義案を作る場合は「AI / JSON Studio」を使います。追加済みの定義は、ここで選べるようになります。
